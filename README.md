## Deployment

To deploy this project run

```bash
  git clone https://github.com/AngeloLT/MiComidaFavorita.git
```

Configurar Firebase

```bash
  src/config/firebase.js
```

Ejecución

Asegurarse de que todas las dependencias estén instaladas:

```bash
  npm install
```

Iniciar la aplicación

```bash
  npx expo start
```
